Antri

Privacy Policy

How Antri handles job-search data.

Effective May 22, 2026

What Antri is

Antri is a job application tracker. The Antri website stores application records for signed-in users, and the Antri Job Saver Chrome extension helps a user create a draft from a job posting they choose to save.

Data you provide

Antri stores the application details you enter or save, such as job role, company, location, dates, status, follow-up details, notes, source, job URL, and contact information. Account authentication may use email/password or Google sign-in.

Chrome extension data

The Antri Job Saver extension only reads the active web page after you click its Save button. For that chosen page, it may collect the job page URL, page title, visible job-page text, and structured job posting data exposed by the page so Antri can create a draft application record.

The extension sends that selected page data over HTTPS to Antri's extraction service. Antri may send job-posting text to an AI extraction provider to turn it into structured application fields. The draft is then opened in Antri for you to review before saving.

How data is used

Antri uses data to authenticate accounts, sync application records, extract job-posting details after a user requests that action, display application tracking views, diagnose service failures, and protect the service from abuse.

Antri does not sell Chrome extension page data and does not use extension page data for advertising.

Service providers

Antri relies on service providers for hosting, authentication, database storage, and AI-assisted extraction. Those providers process data only as needed to operate the Antri service and its requested features.

Google data

If you choose Google sign-in, Antri uses Google authentication information needed to create and access your account. Antri's use and transfer of information received from Google APIs will adhere to the Chrome Web Store User Data Policy, including the Limited Use requirements.

Your choices

You choose whether to enter job application records and whether to run the browser saver on a page. You can edit or delete application records inside Antri. You can remove the extension from Chrome at any time.

Contact

For privacy questions during Antri's beta, contact the Antri operator through the contact channel provided with your beta access.
